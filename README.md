# TicTacToe_Fireworks
A multiplayer X&amp;O game mouse versus keyboard, with some special motion effects
------

Aplicația reprezintă un joc de X și O realizat în Windows Form Aplication, în 
limbaj C#. Aceasta permite doi jucători, un utilizator ce folosește mouse-ul și altul ce folosește 
tastatura, pentru a se duela într-o competiție clasică de Tic Tac Toe.

Interfața se prezintă una simplă, având un form proiectat pentru a fi plăcut ochiului, tema sa 
centrală fiind “Culorile nopții”. Astfel, grilajul de 3X3 realizat din butoane este partea centrală a 
întregii aplicații, având pe fiecare din cele două laterale câte o imagine a cerului înstelat. Un label 
afișează titlul jocului, atât în română, cât și în engleză, Tic Tac Toe X SI O. Sub grilaj sunt poziționate 
alte două labele ce informează utilizatorii de modul de joc de tip Two Players, dar și de dispozitivul 
pe care fiecare jucător îl deține. 

Jucator 1 ClickMouse O

Jucator 2 Tastatura X

Un detaliu aflat în contrast cu elementele anterioare și care atrage atenția utilizatorilor, îl 
reprezintă notele muzicale de culoare roz care se mișcă neîncetat. Dinamismul oferit dă 
viață întregii aplicații pe parcursul jocului. 
Atunci când un jucător reușește să deseneze un model câștigător, se va afișa un mesaj de 
felicitare personalizat pentru acesta, însoțit de un foc de artificii.
În cazul în care toate pătratele sunt 
umplute, dar nici un jucător nu a câștigat, mesajul afișat este cel de Remiză, fără artificii.
